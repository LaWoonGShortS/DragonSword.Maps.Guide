                   1. 사용자 모드 가이드
================================================================================

【1-0. 스플래시 화면】

시작 시 스플래시 화면이 표시됩니다.

닫는 방법:
  - 아무 키나 누르기 (1초 후 활성화)
  - 화면 클릭 또는 터치

【1-1. 기본 조작】

■ 맵 이동
  - 마우스 드래그: 맵 이동
  - 마우스 휠: 확대/축소
  - 더블 클릭: 해당 위치로 확대

■ 마커 조작
  - 마커 클릭: 획득 체크 토글 (반투명 처리)
  - 마커 호버: 툴팁으로 설명 표시

■ 진행도 체크
  - 마커 1번 클릭: 획득 체크 (반투명 처리)
  - 마커 재클릭: 체크 해제
  - 체크된 마커: 브라우저에 자동 저장
  - 브라우저 재시작 후에도 진행도 유지

■ UI 토글
  - ◀ 버튼 (우측): 전체 UI 숨기기/보이기
  - 패널 헤더 클릭: 해당 패널 접기/펼치기

【1-2. 검색 기능】

위치: 오른쪽 사이드바 상단

사용법:
  1. 🔍 검색창에 키워드 입력 (예: "성벽")
  2. 실시간으로 관련 마커만 표시
  3. 검색 결과 개수 하단에 표시
  4. 검색창 비우기 → 전체 마커 복원

【1-3. 필터 기능 (다중 선택 가능)】

위치: 오른쪽 사이드바 (검색창 아래)

사용법:
  - 전체: 모든 마커 표시 (다른 선택 모두 해제)
  - 개별 타입 클릭: 해당 타입 표시/해제
  - 여러 타입 동시 선택 가능!
    예) 마멋왕 + 새알 클릭 → 두 타입 동시 표시
  - 선택된 버튼 다시 클릭 → 해제
  - 모두 해제 시 자동으로 전체 표시

【1-4. 진행도 추적】

위치: 왼쪽 사이드바

표시 정보:
  - 타입별 아이콘 및 이름
  - 현재 획득 수 / 전체 개수
  - 진행률 바 (시각적 표시)
  - 백분율 (%)

초기화:
  - 🔄 전체 초기화 버튼 클릭

【1-5. 좌표 제보】

위치: 오른쪽 사이드바 하단

사용법:
  1. 맵에서 원하는 위치 클릭 → 좌표 자동 입력
  2. 타입 선택
  3. 설명 입력
  4. ➕ 리스트에 추가
  5. 여러 개 추가 가능
  6. 📋 전체 복사 → YouTube 댓글에 붙여넣기

【1-6. 배경음악 플레이어】

위치: 화면 하단 🎵 버튼

기능:
  - 8곡의 드래곤소드 관련 음악
  - ▶️ 재생 / ⏸️ 일시정지
  - ⏮️ 이전곡 / ⏭️ 다음곡
  - 볼륨 조절
  - 곡 목록에서 직접 선택 가능

                      3. 단축키 및 팁
================================================================================

【단축키】

맵 조작:
  - 마우스 드래그: 맵 이동
  - 마우스 휠: 확대/축소
  - 더블 클릭: 빠른 확대
  - Shift + 드래그: 박스 줌 (영역 지정 확대)

일반:
  - Ctrl + Shift + R: 강력 새로고침 (캐시 무시)
  - F11: 전체화면 모드

【팁 & 트릭】

■ 효율적인 수집 루트
  1. 필터로 타입별 분류 (다중 선택 활용!)
  2. 지역별로 순차 수집
  3. 진행도 실시간 확인
  4. 완료된 마커는 자동 체크

■ 검색 활용법
  - 지역 이름으로 검색 (예: "마을", "성", "동굴")
  - 위치 특징으로 검색 (예: "위", "아래", "입구")
  - 부분 검색 가능 (예: "성" → "성벽", "성문")

■ 다중 필터 활용법
  - 마멋왕 + 새알 동시 선택 → 두 타입만 표시
  - 보물상자 + 퍼즐 + 봉인된상자 → 세 타입 동시 표시
  - 원하는 조합으로 자유롭게 필터링

■ 진행도 관리
  - 타입별로 집중 수집
  - 100% 달성 시 다음 타입 진행

■ 좌표 제보 요령
  - 정확한 위치 클릭
  - 명확한 설명 작성 (예: "마을 입구 왼쪽 건물")
  - 여러 개 발견 시 한 번에 제보

■ UI 숨기기 활용
  - 맵만 집중해서 볼 때
  - 스크린샷 찍을 때
  - 모바일에서 화면 확보

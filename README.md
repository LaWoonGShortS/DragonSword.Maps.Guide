================================================================================
                           1. 소개
================================================================================

DragonSWORD Map Guide는 드래곤소드 게임의 모든 수집 요소를 한눈에 확인할 수 
있는 인터랙티브 맵 가이드입니다.

【주요 특징】
• 792개의 마커 데이터 (보물상자, 퀘스트, 숨겨진 요소 등)
• 실시간 검색 및 필터링
• 진행도 자동 추적 및 저장
• 새로운 위치 제보 시스템
• 관리자 모드를 통한 데이터 관리
• 네온 사이버펑크 테마 UI
• 스플래시 화면 및 저작권 표시
• 패널 접기/펼치기 및 UI 숨기기 기능

【기술 스택】
• Leaflet 1.9.4 (맵 라이브러리)
• Vanilla JavaScript (프레임워크 없음)
• HTML5 / CSS3
• LocalStorage (진행도 저장)

================================================================================
                         2. 시작하기
================================================================================

【설치 방법】

1. 저장소 다운로드
   - GitHub에서 ZIP 다운로드 또는 git clone
   - git clone https://github.com/yourusername/dragonsword-map.git

2. 파일 압축 해제
   - dragonsword-map 폴더 생성 확인

3. 필수 파일 확인
   ✓ index.html
   ✓ style.css
   ✓ script.js
   ✓ markers/ 폴더 (7개 JSON 파일)
   ✓ images/dragonsword_map_3000.png

【실행 방법】

방법 1: VSCode Live Server (추천)
   1. VSCode로 프로젝트 폴더 열기
   2. index.html 우클릭
   3. "Open with Live Server" 선택
   4. 브라우저 자동 실행

방법 2: Python 간이 서버
   # Python 3
   python -m http.server 8000
   
   # Python 2
   python -m SimpleHTTPServer 8000
   
   # 브라우저에서 http://localhost:8000 접속

방법 3: Node.js http-server
   npm install -g http-server
   http-server -p 8000
   
   # 브라우저에서 http://localhost:8000 접속

방법 4: 직접 열기
   - index.html 파일을 브라우저로 드래그
   - 또는 더블클릭하여 기본 브라우저로 실행
   - 주의: 일부 기능은 로컬 서버 필요

【시스템 요구사항】
• 모던 웹 브라우저 (Chrome, Firefox, Edge, Safari)
• JavaScript 활성화
• LocalStorage 활성화 (진행도 저장용)
• 최소 화면 해상도: 1280x720
• 권장 화면 해상도: 1920x1080

【4-0. 스플래시 화면】

시작 시 스플래시 화면이 표시됩니다.

표시 내용:
  - 🗺️ DragonSWORD Map 로고
  - 저작권 안내 (Hound13, Webzen)
  - 제작자 정보 (비밀리)
  - "아무 키나 누르거나 클릭하면 시작합니다" 안내

닫는 방법:
  - 아무 키나 누르기 (1초 후 활성화)
  - 화면 클릭 또는 터치

【4-1. 기본 조작】

■ 맵 이동
  - 마우스 드래그: 맵 이동
  - 마우스 휠: 확대/축소
  - 더블 클릭: 해당 위치로 확대

■ 마커 조작
  - 마커 클릭: 획득 체크 토글 (반투명 처리)
  - 마커 호버: 툴팁으로 설명 표시

■ 진행도 체크
  - 마커 1번 클릭: 획득 체크 (반투명 처리)
  - 마커 재클릭: 체크 해제
  - 체크된 마커: 자동으로 LocalStorage에 저장
  - 브라우저 재시작 후에도 진행도 유지

■ UI 토글
  - ◀ 버튼 (우측): 전체 UI 숨기기/보이기
  - 패널 헤더 클릭: 해당 패널 접기/펼치기
  - 상태는 LocalStorage에 저장됨

【4-2. 검색 기능】

위치: 오른쪽 사이드바 상단

사용법:
  1. 🔍 검색창에 키워드 입력 (예: "성벽")
  2. 실시간으로 관련 마커만 표시
  3. 검색 결과 개수 하단에 표시
  4. 검색창 비우기 → 전체 마커 복원

예시:
  - "성벽" 입력 → 성벽 관련 위치만 표시
  - "마을" 입력 → 마을 관련 위치만 표시
  - "보물" 입력 → 보물 관련 위치만 표시

【4-5. 좌표 제보】

위치: 오른쪽 사이드바 하단

사용 시나리오:
  - 새로운 위치 발견
  - 데이터베이스에 없는 마커
  - 관리자에게 제보하고 싶을 때

사용법:
  1. 맵에서 원하는 위치 클릭
     → X, Y 좌표 자동 입력
     → 알림: "📍 좌표: (X, Y)"
  
  2. 타입 선택
     - 드롭다운에서 타입 선택
     - 🧰 보물상자, 🦫 마멋왕 등
  
  3. 설명 입력
     - 예: "성벽 위", "마을 입구", "동굴 안"
  
  4. ➕ 리스트에 추가 버튼 클릭
     - 리스트에 추가됨
     - 여러 개 추가 가능
  
  5. 📋 전체 복사 버튼 클릭
     - 텍스트 형식 + JSON 형식 동시 복사
     - 클립보드에 복사됨
  
  6. YouTube 댓글 또는 커뮤니티에 제보
     - Ctrl+V로 붙여넣기

복사 형식 예시:

     좌표: (45.23, 67.89)

      "description": "성벽 위",

팁:
  - 여러 위치 발견 시 한 번에 추가 후 복사
  - 🗑️ 삭제 버튼으로 개별 삭제 가능

접을 수 있는 패널:

  - 🔍 위치 검색

  - 📍 새 좌표 제보

상태 저장:
  - LocalStorage에 자동 저장
  - 페이지 새로고침 후에도 상태 유지

위치: 화면 우측 (관리자 패널 옆)

【5-1. 관리자 모드 활성화】

접속 방법:
  1. 화면 좌측 상단 "⚙️ 관리자" 버튼 클릭
  2. 비밀번호 입력창 표시
  3. 비밀번호 입력: ****
  4. "확인" 클릭
  5. 관리자 패널 활성화

비밀번호 변경 방법:
  - script.js 파일 열기
  - initModeSwitch 함수에서 찾기: if (pw !== '****')
  - '****'을 원하는 비밀번호로 변경
  - 저장 후 새로고침

주의사항:
  - 관리자 모드는 데이터 수정 권한 포함
  - 비밀번호 유출 주의
  - 프로덕션 환경에서는 서버 측 인증 권장

【5-2. 새 마커 추가】

방법:
  1. 관리자 모드 활성화
  2. 맵에서 원하는 위치 클릭
  3. 🧰 보물상자 마커 자동 생성
  4. 편집 팝업 자동 열림
  5. 타입 선택 (드롭다운)
  6. 설명 입력
  7. 💾 저장 버튼 클릭

편집 팝업 내용:
  - 🆕 새 마커 편집 (제목)
  - 타입 선택 (드롭다운)
  - 설명 입력란
  - 현재 좌표 표시
  - 💾 저장 버튼

마커 속성:
  - isNew: true (새 마커 표시)
  - draggable: true (드래그 가능)
  - 추가 마커 목록에 자동 등록

【5-3. 기존 마커 이동】

방법:
  1. 관리자 모드 활성화
  2. 이동할 마커 클릭 후 드래그
  3. 원하는 위치로 드래그
  4. 마우스 버튼 놓기
  5. 자동으로 "이동됨" 목록에 추가

이동 정보 표시:
  - 📍 이동됨 섹션에 표시
  - 이전 좌표와 현재 좌표 비교
  - 타입, 이름, 설명 포함

주의사항:
  - 기존 마커만 이동 추적됨
  - 새 마커 이동은 추가 목록에서만 업데이트
  - 드래그 중 좌표 실시간 업데이트

【5-4. 마커 삭제】

삭제 가능 조건:
  - 새로 추가한 마커만 삭제 가능
  - 기존 마커는 삭제 불가 (데이터 보호)

방법:
  1. 관리자 모드 활성화
  2. 삭제할 새 마커에 Alt 키 누른 상태로 클릭
  3. 마커에 빨간 효과 표시 (선택됨)
  4. 다중 선택 가능 (여러 마커 Alt+클릭)
  5. 🗑️ 선택 삭제 버튼 클릭
  6. 확인창: "🗑️ 선택된 N개 마커를 삭제할까요?"
  7. "확인" 클릭 → 삭제 완료

선택 표시:
  - 빨간색 그림자 효과
  - 흔들림 애니메이션
  - 반투명 처리 (opacity: 0.7)

A4: 해결 방법

    1. 관리자 모드 활성화 확인
    2. "⚙️ 관리자" 버튼 클릭

    4. 관리자 패널 표시 확인

A6: 확인 사항
    1. 관리자 모드 활성화 확인

A7: 좌표 시스템 확인

    - 픽셀 좌표: 0-3000 범위
    - 변환 공식: 픽셀 = (맵좌표 / 100) × 3000
    
    예시:
    - 맵 좌표 (50, 50) = 픽셀 (1500, 1500) = 맵 중앙
    - 맵 좌표 (0, 0) = 픽셀 (0, 0) = 좌측 상단
    - 맵 좌표 (100, 100) = 픽셀 (3000, 3000) = 우측 하단

    3. 화면 아무 곳이나 클릭
    
    확인 사항:
    - 1초 보호 시간이 있음
    - JavaScript 활성화 확인
    - 콘솔에서 에러 확인
